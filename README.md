# Task1-1-Data-Cleaning
Data Cleaning and Preprocessing using Python (Pandas)
Task 1: Data Cleaning and Preprocessing

Objective

To clean and preprocess the Netflix Movies and TV Shows dataset using Python and Pandas.

Dataset

- Netflix Movies and TV Shows ("netflix_titles.csv")

Tools Used

- Python
- Pandas
- Visual Studio Code
- GitHub

Steps Performed

- Loaded the dataset.
- Displayed the first 5 rows.
- Checked missing values.
- Removed duplicate rows.
- Standardized column names.
- Filled missing values in "director", "cast", "country", "rating", and "duration".
- Converted the "date_added" column to datetime format.
- Saved the cleaned dataset as "netflix_titles_cleaned.csv".

Files Included

- "clean_data.py"
- "netflix_titles.csv"
- "netflix_titles_cleaned.csv"
- Screenshots

Result

The dataset was successfully cleaned and prepared for further analysis.
